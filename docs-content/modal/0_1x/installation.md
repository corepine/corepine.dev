# Installation

Set up Corepine Modal inside your Laravel project.

## Requirements

- PHP 8.3+
- Laravel 11+
- Tailwind-based frontend stack

## Install via Composer

```bash
composer require corepine/modal
```

## Publish Config

```bash
php artisan vendor:publish --tag=corepine-modal-config
```

## Verify Installation

Run a quick smoke test by rendering your first modal from the [Quick Start](doc:quick-start).
